# Sudoku-App

Kişisel kullanım için geliştirilen sade, hızlı ve yüksek zorluk seviyelerine odaklanan Sudoku uygulaması.

## Dosya yapısı

Güncellemeleri kolaylaştırmak için uygulama üç küçük dosyaya bölündü:

- `index.html`: Sayfa iskeleti, CSS/JS bağlantıları ve uygulama kökü
- `styles.css`: Görsel tasarım, responsive yerleşim ve yürüyen kedi animasyonu
- `app.js`: Sudoku üretimi, oyun mantığı, timer, not modu, ipucu ve kayıt sistemi

## En kolay yayınlama

GitHub reposuna bu üç dosyayı kök dizine koyman yeterli. Vercel statik dosyaları doğrudan yayınlayabilir.

## Özellikler

- Bağımlılıksız HTML, CSS ve JavaScript uygulaması
- Kireçte Troçki adı, zaman sayacı ve yürüyen siyah kedi detayı
- Uzman, Ekstrem ve Kâbus zorluk seviyeleri
- Pasif not alma modu
- Seçili hücre, satır, sütun, kutu ve aynı sayı vurguları
- Sınırlı ipucu mekanizması
- Hata sayacı, geri alma ve yerel kayıt
